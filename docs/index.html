ESP32 Water Meter

Install the firmware from your browser.

Use this page to flash the ESP32 water meter without PlatformIO, Arduino IDE, or a local toolchain. After installation, the device exposes its own Web UI for configuration, calibration, and OTA updates.

One-click installer

Click install, connect the ESP32 over USB, and select the port when your browser asks. Chrome, Edge, and other Chromium-based browsers are supported.

What gets flashed

The installer publishes the ESP32 bootloader, partition table, app binary, and the runtime Web UI firmware.

After install

Open the device on your LAN, use the Web UI to set Wi-Fi and MQTT, then save settings and reboot.

Need the sensor wired?

This firmware expects the LIS3MDL on GPIO18, GPIO19, GPIO23, and GPIO5.

GitHub Pages must be enabled for this repository, and the browser must use HTTPS for Web Serial access.
