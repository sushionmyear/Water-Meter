Browser Installer

Water Meter ESP32

Connect the ESP32 by USB, close any serial monitors, and click install. This writes the complete factory image for a blank board.

  • Blank board friendly
  • No PlatformIO required
  • Future firmware updates can use OTA

Before You Start

  • Use a USB data cable, not a charge-only cable.
  • Power the LIS3MDL from 3.3V.
  • Make sure no other app is using the serial port.

After Flashing

  1. Open the serial monitor at 115200.
  2. Find the device IP address after Wi-Fi connects.
  3. Open http://<device-ip>/.
  4. Finish setup in the Web UI.

What You Can Configure

  • Wi-Fi and MQTT settings
  • Meter calibration and thresholds
  • Pulse timing and persistence
  • OTA firmware updates

If the install button is unavailable, use the README browser installer link from GitHub or flash the ESP32 once over USB with PlatformIO and use OTA after that.
